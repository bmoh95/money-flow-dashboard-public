# 학원 여름방학 특강 안내문 무료 검증 브리프

## 스타일 샘플 요약
- 샘플 20개 검색: 학원 방학특강 안내문, 학원 공지 문자, 학원 상담 문자, 보강 안내 문자.
- 제목 패턴: “학원 ○○ 안내문/문자 예시”, “방학특강 모집 전 준비”, “공지 문자 샘플 N종”.
- 첫 문단: 원장님이 바로 겪는 상황(학부모 문의, 일정 확정, 정원 마감)을 짧게 언급.
- 문단 길이: 1~3문장, 예시는 박스/인용문으로 분리.
- 이미지 빈도: 안내문 카드/문자 캡처/체크리스트 2~4장 필요.
- CTA: 강하지 않음. “반응 있으면 상황별 문구를 더 정리” 정도가 자연스러움.

## 스타일 브리프
- 대상 독자: 여름방학 특강을 준비하는 소형 보습학원·영어/수학학원 원장님.
- 목소리: 학원 운영을 옆에서 본 실무자 말투. 과장 없이, 바로 복붙 가능한 예시 중심.
- 피할 것: 교육 성과 보장, 과한 마케팅 문구, 법률/입시 자문처럼 보이는 단정, 완벽한 AI식 목차.
- 이미지 계획:
  1. 썸네일: “여름방학 특강 안내문, 이렇게만 정리해도 덜 어색합니다”
  2. 문자 캡처 카드: 특강 모집 1차 안내 예시
  3. 체크리스트 카드: 일정·정원·보강·환불/변경 안내 점검표
- 소프트 CTA: “학원별 상황이 달라서, 반응 있으면 상담/마감/보강 문구도 따로 정리해볼게요.”

## 무료→유료 경로
- 무료 글: 특강 안내문/문자 구조와 예시 8개 제공.
- 후속 유료 가능성: 학원 상담 문자·특강 모집·보강 안내·마감 임박·결제 리마인드 템플릿 ZIP.
- 검증 기준: Tistory 유입, 검색 노출, 저장/댓글/추가 요청 여부.
