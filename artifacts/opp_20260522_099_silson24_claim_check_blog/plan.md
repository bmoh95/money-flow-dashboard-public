# Plan — 실손24 청구 전 확인 3가지 무료 검증 글

## 제목 후보

1. `실손24, 병원 안 가도 된다더니 먼저 확인할 3가지`
2. `실손보험 청구 간소화, 내 병원은 되는지 먼저 보세요`
3. `동네 의원·약국 실손24, 서류 떼기 전에 참여병원부터 확인`

## 독자/채널

- 독자: 실손보험 가입자, 가족 대신 청구하는 보호자, 동네 의원·약국 진료 후 청구 방법을 찾는 사람.
- 채널: Tistory 무료 검증 글. 판매 CTA 없음.
- 카테고리 후보: 돈관리·구매체크 또는 뉴스·시사 팩트체크. 발행 전 실제 categoryId 확인.

## 본문 구조

1. 첫 문단: “서류 떼러 병원 가지 않아도 된다”는 말이 맞는 경우와 아닌 경우가 있음을 바로 제시.
2. 질문 3개:
   - 내가 다녀온 병원/약국이 실손24 참여기관인가?
   - 내 보험사가 실손24 청구를 지원하는가?
   - 연계가 안 되면 기존 서류 청구로 가야 하나?
3. 공식 보도자료 핵심: 전체 10.4%, 의원·약국 6.9% 연계율이라 참여기관 확인이 먼저.
4. 실손24 참여병원 검색 순서: 병원이름 입력 → 참여병원 표시 확인 → 없으면 기존 청구 준비.
5. 오해 방지: 자동 지급 아님, 보험금 지급 여부 단정 아님, 개인정보는 공식 실손24/보험사에서만.
6. 공식 링크: 금융위 보도자료, 실손24 참여병원 검색, 정책브리핑.

## 품질/차별화

- 기존 글의 `자동청구 총정리` 과장을 피하고, 독자가 바로 확인할 참여기관 검색을 첫 화면에 둠.
- 공식 연계율 수치를 넣어 “모든 의원·약국 가능” 오해를 줄임.
- 보험 상담/권유가 아니라 청구 경로 확인 안내로 한정.
