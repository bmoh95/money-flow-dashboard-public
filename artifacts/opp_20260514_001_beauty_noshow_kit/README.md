# 뷰티샵 노쇼·당일취소 방지 예약 운영 키트 KR

네일샵·속눈썹샵·왁싱샵·1인 헤어샵 원장님이 **오늘 5분 안에 복붙해서 쓰는** 예약금/리마인드/취소응대/빈타임 재판매 운영 패키지입니다.

## 구성
1. `01_kakao_sms_templates_90.md` — 상황별 카톡/SMS 문구 90개
2. `02_reservation_policy_builder.md` — 예약금·지각·당일취소·환불 규정 빌더
3. `03_no_show_response_playbook.md` — 노쇼/지각/반복취소 대응 플레이북
4. `04_daily_reservation_ops_tracker.csv` — 예약 운영 트래커
5. `05_empty_slot_resale_calendar.csv` — 빈타임 재판매 캠페인 캘린더
6. `06_sales_page.md` — 판매 페이지 원고
7. `07_upload_copy.md` — 포스타입/크몽/스마트스토어 업로드 문구
8. `08_usage_guide.md` — 5분 적용 가이드
9. `09_market_validation.md` — 시장성·경쟁·마진 검증 근거

## 5분 사용법
1. `02_reservation_policy_builder.md`에서 샵에 맞는 정책 1개를 고릅니다.
2. 카카오채널/예약폼/인스타 하이라이트에 정책을 붙입니다.
3. `01_kakao_sms_templates_90.md`의 “예약 확정”, “D-1 리마인드”, “당일 지각” 문구를 복사합니다.
4. `04_daily_reservation_ops_tracker.csv`로 취소 사유와 회수 여부를 기록합니다.

## 주의
- 법률 자문이 아닙니다. 사업장 정책·지역·플랫폼 약관에 맞게 조정하세요.
- 소비자에게 예약 전 명확히 고지하고, 증빙 가능한 채널에 남기는 방식으로 사용하세요.
- 수익 보장 표현 없이 운영 손실을 줄이는 실무 도구로 판매하세요.
