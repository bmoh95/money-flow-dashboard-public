# Plan — 첫 직원 채용 후 4대보험 사업장 성립신고 체크 글

## 한 줄 scope
첫 직원이나 알바를 처음 뽑은 사장님이 `사업장 성립신고`, `근로자 자격취득신고`, `원천세/급여 자료`를 헷갈리지 않게 순서로 보는 무료 Tistory 글.

## 독자
- 첫 직원을 채용한 개인사업자
- 알바를 처음 쓰는 작은 카페·학원·공방·매장 운영자
- 4대보험 정보연계센터 화면에 들어가기 전 무엇을 준비할지 모르는 사장님

## 채널/카테고리
- Tistory `돈관리·구매체크 > 세금·장부` 우선. 직원 운영 카테고리가 별도 없으므로 노무/원천세와 연결.
- 무료 글. 판매 CTA 없음.

## 제목 후보
1. 첫 직원 뽑았다면, 4대보험 신고 전에 이 순서만 먼저 보세요
2. 개인사업자 첫 직원 채용, 사업장 성립신고부터 헷갈리는 것만 정리
3. 알바 처음 쓰는 사장님이 4대보험에서 놓치기 쉬운 5가지

## 차별화
보험 제도 설명을 길게 하지 않고 `직원 뽑은 날 → 신고 전 준비 → 신고 후 확인` 타임라인으로 정리한다. 대표자 가입 여부, 직원 근로시간, 급여자료 메모처럼 실수 지점을 앞에 둔다.

## 이미지 계획
1. Thumbnail: `첫 직원 채용 / 4대보험 순서` / 매장 사장님+체크 문서 아이콘 / 인디고·민트.
2. Explainer: 채용일 → 근로조건 확인 → 사업장 성립 → 자격취득 → 급여/원천세 메모 흐름.
3. Action card: 신고 전 체크 5개 — 채용일, 근로시간, 월급/시급, 대표자 구분, 공식사이트.

## 실행 체크리스트
- 4대사회보험 정보연계센터와 찾기쉬운 생활법령 문구 재확인.
- 신고기한·가입대상은 단정 대신 공식 확인 링크와 `상황별 차이`로 표현.
- 최근 세금 글의 `한눈에 보면` 반복을 피하고 타임라인/실수 사례 구조 사용.
- 이미지 3장 생성 및 local visual QA.
- Tistory 카테고리·HTML 렌더링·공개 visual QA는 backlog job으로 넘김.

## ROI 가정
- 보수: 0원. 즉시 매출 없음.
- 기준: 첫 채용/4대보험 검색 유입 누적, 장기 기대가치 34,000원 상당.
- 낙관: 직원 운영·급여 체크 콘텐츠군으로 내부링크 확장.
