# Plan — 수산대전 6월 온누리상품권 환급 확인 글

## 제목 후보
1. 6월 수산대전 환급, 장보기 전 3가지만 보세요
2. 수산물 사면 최대 2만원? 6월 온누리 환급 확인 순서
3. 6월 10~14일 수산대전 환급, 내 시장도 되는지 먼저 보기

## 대상 독자
- 6월 10~14일 수산시장·전통시장 방문 예정자
- 국내산 수산물 구매 후 온누리상품권 환급을 놓치고 싶지 않은 독자
- “내 시장도 되나, 얼마 사야 하나, 영수증은 어디로 가나”를 빠르게 보고 싶은 독자

## 글 구조
1. 첫 화면: 5초 판정 — 날짜, 환급 구간, 확인 순서.
2. 내 경우는? — 참여시장/점포 확인 → 국내산 수산물 구매 → 영수증/본인확인 → 부스 환급.
3. 금액표 — 3만4천원 이상 1만원, 6만7천원 이상 2만원. 단, 시장별 공지 우선.
4. 헷갈리는 부분 — 온누리상품권 구매 행사가 아니라 구매 후 환급행사, 당일 영수증, 참여점포 차이.
5. 공식 링크 — 대한민국 수산대전, 수원시 예시 보도자료, 각 시장 공지 확인 안내.
