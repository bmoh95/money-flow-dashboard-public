# 종합소득세 신고 끝? 개인지방소득세까지 확인해야 진짜 끝입니다

종합소득세 신고를 끝내고 나면 보통 바로 창을 닫습니다.

저도 예전엔 “홈택스에서 접수됐으면 끝난 거 아닌가?” 하고 생각했는데요. 5월 신고 때는 하나를 더 봐야 합니다.

바로 **개인지방소득세**입니다.

[이미지 1: 종합소득세 끝? 지방소득세 확인 제목 카드]

## 한눈에 보면

- 종합소득세는 국세입니다.
- 개인지방소득세는 지방세입니다.
- 그래서 신고·납부 확인 화면이 따로 보일 수 있습니다.
- 홈택스 신고 후 위택스나 지자체 안내 화면까지 확인하는 게 안전합니다.

어렵게 말하면 헷갈리는데, 쉽게 말하면 이렇습니다.

“종소세 신고했으니 끝”이 아니라, “지방소득세도 접수·납부 상태가 맞는지 한 번 더 확인”입니다.

## 어디서 많이 놓치냐면

모두채움 안내문을 보고 그대로 신고한 분들이 특히 헷갈립니다.

홈택스에서 종합소득세 접수는 했는데, 개인지방소득세 납부서나 환급 안내를 제대로 못 보고 지나가는 경우가 있습니다.

금액이 작아 보여도 신고 상태가 애매하면 나중에 더 귀찮아집니다.

## 확인 순서

[이미지 2: 홈택스 신고 → 위택스 확인 → 납부/환급 확인 흐름 카드]

1. 홈택스에서 종합소득세 신고 접수 여부를 확인합니다.
2. 개인지방소득세 신고로 이어지는 안내가 있었는지 봅니다.
3. 위택스 또는 지자체 안내 페이지에서 신고·납부 상태를 다시 확인합니다.
4. 납부할 금액이 있으면 납부기한을 확인합니다.
5. 환급이면 계좌와 처리 안내를 확인합니다.

여기서 중요한 건 “내가 신고했다는 기억”보다 “화면에서 접수/납부 상태가 확인되는지”입니다.

## 사장님들이 자주 헷갈리는 부분

### 1. 종합소득세와 지방소득세가 같은 세금인가요?

아닙니다.

종합소득세는 국세, 개인지방소득세는 지방세라서 담당 시스템과 고지 흐름이 다를 수 있습니다.

### 2. 종소세 환급이면 지방소득세도 무조건 환급인가요?

케이스마다 다릅니다.

환급/납부 여부는 실제 신고 결과와 안내 화면을 기준으로 확인해야 합니다.

### 3. 납부 금액이 작으면 그냥 넘어가도 되나요?

그렇게 판단하면 안 됩니다.

금액보다 중요한 건 신고·납부 상태가 정상인지입니다.

## 마지막 체크표

[이미지 3: 개인지방소득세 마지막 체크리스트 카드]

| 확인할 것 | 봐야 할 내용 |
|---|---|
| 종합소득세 접수 | 홈택스 접수 완료 여부 |
| 지방소득세 신고 | 위택스/지자체 화면에서 신고 상태 |
| 납부 여부 | 납부할 금액과 기한 |
| 환급 여부 | 환급 계좌와 안내 문구 |
| 증빙 저장 | 접수증, 납부서, 안내 화면 캡처 |

저라면 최소한 접수증이나 납부서 화면은 PDF나 캡처로 따로 저장해둡니다.

나중에 “했나?” 싶을 때 제일 빨리 확인할 수 있어서요.
