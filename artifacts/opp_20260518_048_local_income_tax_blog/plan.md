# Plan — 개인지방소득세 체크 무료 검증 글

## 독자
5월 종합소득세를 홈택스에서 끝냈다고 생각한 1인 사업자·프리랜서.

## 채널
Tistory `돈관리·구매체크 > 세금·장부` 무료 정보 글. 판매 CTA 없음.

## 차별점
대부분 글은 신고 방법을 길게 설명한다. 이 글은 “종합소득세 신고 후 아직 남은 1개 확인”으로 좁히고, 독자가 위택스/지자체 안내에서 상태를 확인하도록 체크리스트화한다.

## 글 구성
1. 종소세 끝났다고 바로 닫기 전에 확인할 것
2. 개인지방소득세가 왜 따로 보이는지
3. 홈택스 신고 후 위택스/지자체 신고·납부 상태 확인 순서
4. 놓치기 쉬운 5가지: 납부서, 환급, 가산세, 모두채움, 납부기한
5. 사장님용 마지막 체크표
6. 공식 링크 모음과 세무사 확인 권장 문구

## 이미지 계획
- 01 썸네일 PNG: `종합소득세 끝? / 지방소득세 확인`
- 02 흐름 카드 PNG: `홈택스 신고 → 지방소득세 확인 → 납부/환급 확인`
- 03 체크리스트 PNG: `납부기한·납부서·환급계좌·신고상태`

## ROI 가정
- 직접 매출: 0원. 콘텐츠 판매 중단 상태라 무료 검증 전용.
- 기준 순가치: 기존 세금 카테고리 축적·검색 유입 검증 자산. 제작비 0원, 외부 게시만 backlog job에서 처리.
- 보수/기준/낙관: 0원 / 0원 직접수익+검색 검증 / 애드센스·향후 도구 전환 가능성.
