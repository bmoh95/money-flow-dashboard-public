# 이미지 계획

1. 썸네일/판정 카드
   - 문구: `서울 청년수당 2차` / `5월29일 16시 전 확인`
   - 포인트: 월 50만원, 최대 6개월, 서울 거주 미취업자.
   - 색감: 민트/파랑, 시계+체크 아이콘. Pretendard Bold/SemiBold.

2. 구조 카드
   - 제목: `먼저 걸러볼 3가지`
   - 내용: 서울 거주 / 만19~34세 / 최종학력 졸업·미취업.
   - 목적: 첫 화면 즉시 효용. 작은 글씨 남발 금지.

3. 행동 카드
   - 제목: `신청 전 순서`
   - 내용: 공식 공고 확인 → 청년몽땅정보통 접속 → 마감시간 전 제출.
   - 주의: 비공식 링크/대행 유도 금지.
