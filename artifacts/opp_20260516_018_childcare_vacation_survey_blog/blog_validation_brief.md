## 이번 스카우트 후보 4개
1. GO/HOLD: 어린이집·유치원 여름방학 등원 수요조사 안내문/문자 — 실제 블로그 샘플, 학교/어린이집 공지, 네이버 프리미엄콘텐츠·해피캠퍼스·예스폼 등 유료/무료 대체재 확인. 무료 Tistory 검증 글로 범위 제한.
2. NO-GO: 장마철 배달 지연 안내문 — 식당·쇼핑몰 pain은 있으나 기존 카페/장마 후보와 일부 중복, 음식 변질·배상 표현 리스크가 커 이번 회차 보류.
3. HOLD: 소상공인 개인정보처리방침 템플릿 — 공식 법률/개인정보 규정 확인이 더 필요하고 법률문서 오해 위험이 높아 무료 정보글 전환 전 리스크 큼.
4. HOLD: AstroxNetwork agent_dart API docs bounty — 공식 GitHub 이슈와 20 ICP 보상 확인. 다만 ICP payout/assignment/중복 작업 규칙이 불명확하고 SDK 파악 시간이 커 단기 ROI 낮음.

## 샘플/수요 신호 14개
- DuckDuckGo `어린이집 유치원 여름방학 등원 수요조사 안내문 문자 블로그`: 실제 네이버 블로그/티스토리 양식 글 다수 노출.
- “유치원 여름방학 편지 & 어린이집 여름방학 수요조사서 샘플” 같은 실무형 블로그 글 확인.
- “여름방학 등원 수요조사 안내문 - 네이버 프리미엄콘텐츠”가 노출되어 paid comparable 존재.
- “여름방학 수요조사서 / 여름 가정학습 등원 조사서” 실제 블로그 자료 노출.
- 중앙육아종합지원센터 문서자료실 검색 결과가 있어 어린이집 문서 양식 수요의 공식 채널 신호 존재.
- 대학교 부설어린이집의 “여름 방학 기간 등원 수요조사” 실제 공지 확인.
- 해피캠퍼스 `어린이집 여름방학수요조사서 서식` 노출로 유료 서식 대체재 존재.
- 예스폼 `여름 방학안내문(어린이집)` 노출로 문서형 대체재 존재.
- Kidkids/찬진교육 등 유아교육 콘텐츠 플랫폼 검색 결과 존재.
- 실제 글들은 편지형 인사말 + 기간 + 등원 희망 여부 + 제출기한 + 준비물 안내 흐름이 많음.
- 학부모에게는 “방학인데 왜 수요조사?” 설명이 중요하고, 교사에게는 짧은 키즈노트/문자 문구가 필요.
- 개인정보는 아이 이름/반/보호자 연락처를 예시 본문에 과하게 넣지 않고 `{반}`, `{이름}` 변수 처리해야 함.
- 유료화는 바로 판매보다 여름방학/가정학습/긴급보육/방과후 과정별 안내문 묶음으로 검증 후 판단.
- 무료 글은 안내문 3개, 키즈노트 문구 5개, 수요조사 체크리스트 1개가 적합.

## 스타일 브리프
- 대상 독자: 여름방학 전 수요조사 안내문을 급히 정리해야 하는 어린이집·유치원 원장님/교사.
- 목소리: 교실 일정표와 학부모 답변을 같이 보는 담임 교사 말투. 차분하고 실제적.
- 피할 것: 행정 매뉴얼식 장문, 교육 효과 과장, 개인정보 예시 노출, 학부모 탓하는 표현, 딱딱한 AI 목차.
- 이미지 계획:
  1. 썸네일: “여름방학 등원 수요조사, 이렇게 쓰면 덜 헷갈립니다”
  2. 키즈노트 예시 카드: 짧은 안내문 3문장
  3. 체크리스트 카드: 기간·제출기한·등원일·준비물·급식/차량 여부
- CTA: “반응이 있으면 가정학습/긴급보육/방과후 과정 문구도 따로 정리해볼게요.”
