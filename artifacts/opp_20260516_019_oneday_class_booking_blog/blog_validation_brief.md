# 공방 원데이클래스 예약 안내문 무료 검증 브리프

## 기존 작업 빠른 재점검
- MF-1 `beauty_noshow_kit`: Tistory 무료 공개 완료. 재작성/재게시/승인 요청 없음.
- 기존 launched/tracking: 새 수익·병합·트래픽 수치 없음.
- 기존 redteam_hold 블로그 후보: 이미지·렌더링 QA 대기. 이번 실행은 신규 항목 생성 우선.

## 이번 스카우트 후보 4개
1. GO/HOLD: 공방·원데이클래스 예약 안내/변경/준비물 공지 — 실제 네이버/티스토리 공방 공지 다수, 솜씨당/움클래스/모카클래스 등 클래스 채널 신호, 안내문·문자 템플릿 수요 확인. 무료 Tistory 검증 글로 범위 제한.
2. NO-GO: 사진관 여름휴가 촬영 예약 변경 안내문 — 실제 예약/환불 규정 글은 있으나 사진관별 계약·보정·촬영권 조건이 복잡해 범용 글 가치가 약함.
3. HOLD: 소상공인 개인정보처리방침 템플릿 — 반복 수요는 있으나 법률문서 오해와 공식 조항 검토 부담이 큼.
4. HOLD: Algora 문서화 bounty — bounty 채널은 실제이나 assignment/payout/중복 작업 확인 전 단기 ROI 낮음.

## 샘플/수요 신호 12개
- DuckDuckGo `원데이클래스 예약 안내 주의사항 공방 블로그`: “원데이클래스 예약안내 및 주의사항 공지” 등 실제 공방 공지 다수.
- “원데이클래스 예약 완벽 가이드 주의사항과 팁” Tistory 검색 노출.
- 네이버 블로그 `원데이클래스 수강시 주의 사항`, `[예약안내] 원데이클래스 예약안내사항` 노출.
- `공방 원데이클래스 예약금 환불 안내 공지`: 실제 공방 환불 규정 안내글 다수.
- 헷 은공방 Tistory `클래스 예약 문의 및 환불 규정 안내` 노출.
- 움클래스/솜씨당/모카클래스 등 원데이클래스 중개·B2B 채널 존재.
- Canva 안내문 템플릿, 예스폼 안내문 양식, 예약 취소 문자 템플릿 검색 노출.
- 실제 글 흐름은 예약방법 → 예약금/입금 → 변경·취소 기준 → 준비물/소요시간 → 늦을 때 연락 순서가 많음.
- 독자 pain: DM/카톡으로 같은 질문 반복, 노쇼/지각/재료 준비 손실, 너무 강한 환불 문구에 대한 부담.
- 법적 주의: 환불/위약금 기준은 업장·플랫폼 약관과 소비자 기준에 따라 다를 수 있어 예시는 “운영 안내 문구”로만 제공.
- 무료 글은 예약 안내문 3종, 카톡 짧은 문구 6개, 발송 전 체크리스트 1개가 적합.
- 후속 유료화는 공방 업종별(도자기/향수/가죽/베이킹/반지) 예약 안내 ZIP으로 검증 후 판단.

## 스타일 브리프
- 대상 독자: 혼자 DM·카톡 예약을 받는 공방 대표/원데이클래스 운영자.
- 목소리: 예약 문의를 직접 받아본 공방 사장님 말투. 단호하지만 손님 탓하지 않음.
- 피할 것: 법률 자문처럼 보이는 환불 단정, 플랫폼 약관 베끼기, 광고문, 완벽한 목차, AI식 결론.
- 이미지 계획:
  1. 썸네일: “원데이클래스 예약 안내, 이 5가지는 먼저 써두세요”
  2. 카톡 예시 카드: 예약 확정/변경 문의 답장
  3. 체크리스트 카드: 날짜·인원·입금·소요시간·준비물·변경 기준
- CTA: “반응이 있으면 공방 업종별 문구도 따로 정리해볼게요.”

## 무료→유료 경로
- 무료 글: 바로 복사해 고칠 수 있는 예약 안내/변경/준비물 문구 제공.
- 후속 ZIP: 도자기·향수·가죽·베이킹·반지 공방별 예약/취소/지각/준비물/단체문의 문구 100개.
- 검증 기준: Tistory 검색 유입, 공방/원데이클래스 예약안내 키워드 유입, 댓글/문의 반응.
