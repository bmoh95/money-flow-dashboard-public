# Plan — 지원사업 서류에 ‘소상공인확인서’가 나오면 먼저 볼 4가지

## Scope
Free Tistory validation post only. No paid ZIP/template CTA.

## Buyer / reader
정책자금, 지원사업, 입점·계약 서류를 준비하는 개인사업자/소상공인.

## Channel
Tistory `검증노트` 무료 글. 추천 카테고리: `돈관리·구매체크 > 세금·장부`.

## Differentiator
기존 검색 결과는 발급 절차 중심이다. 이 글은 제출 전 실수 방지: `서류명 → 사업자 정보 → 유효기간 → 제출처 파일형식` 판단 순서로 좁힌다.

## Execution checklist for backlog job
1. 정부24/중소기업현황정보시스템 공식 문구 재확인.
2. 1200×675 PNG 이미지 3장 생성: 썸네일, 흐름도, 체크카드.
3. 이미지 내부 Korean Gothic font/spacing/contact-sheet QA.
4. Tistory 본문 HTML에 이미지 CDN URL 치환.
5. 카테고리 `돈관리·구매체크 > 세금·장부` 지정.
6. 공개 렌더링/모바일/OG/list thumbnail QA 후 tracking 전환.

## ROI assumptions
- Conservative: 0원 direct revenue.
- Base: 25,000원 equivalent strategic value from one durable 행정서류 검색 자산 + internal-link reuse.
- Optimistic: 75,000원 equivalent if it becomes a support-doc cluster entry point.
- Platform fee/tool cost: 0원. User involvement: 0 until backlog publication, which has standing L3 only after QA.
