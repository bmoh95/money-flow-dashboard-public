# 세탁소·셀프빨래방 장마철 이불/이염/분실 문의 안내문 무료 검증 브리프

## 기존 작업 빠른 재점검
- MF-1 `beauty_noshow_kit`: Tistory 무료 공개 완료. 재작성/재게시 없음.
- `smartstore_review_kit`, `changelog_bounty`: 추적 상태. 새 공개 수익/병합 신호 확인 없음.
- 기존 `redteam_hold` 블로그 후보들은 이미지·렌더링 QA 대기. 이번 hourly run은 신규 작업 항목 생성 우선.

## 이번 스카우트 후보 4개
1. GO/HOLD: 세탁소·셀프빨래방 장마철 이불세탁, 이염·분실 접수 안내문 무료 검증 글 — 장마철 검색 수요와 소비자 분쟁 pain이 명확.
2. PIVOT: 세탁소 클레임 대응 문구 유료 ZIP 직행 — 법적/보상 단정 리스크가 있어 무료 운영 안내 글로 축소.
3. NO-GO: 안내문 디자인 외주/템플릿 리셀 — Canva/미리캔버스/크몽 대체재가 강해 차별성 약함.
4. NO-GO: 유료 원고료 블로그 기자단 — 지급처·권리 조건·국내 정산 경로가 불명확하고 사용자 프로필/계정 부담이 큼.

## 샘플/수요 신호 10개
- DuckDuckGo: “세탁소 이염 분실 보상 안내문” — Tistory 소비자 보상/분쟁 글 다수 노출.
- DuckDuckGo: “빨래방 장마철 이불 세탁” — 네이버 매장/이용후기 글 다수 노출.
- Tistory: 세탁소 옷 분실/손상 보상 현실 정리 — 소비자 불만과 분쟁 검색 신호.
- Tistory: 세탁물 사고 소비자분쟁해결기준 분석 — 법/보상 단정 표현 위험 신호.
- Naver Blog: 장마철 셀프빨래방 이용 후기/매장 홍보 글 — 계절 수요.
- Naver Blog: 범어동 24시 빨래방 장마철 이불 수거배달 — 서비스형 세탁 수요.
- Naver Blog: 이지워시 문지동 장마철 대비 — 매장 공지/홍보 톤 참고.
- 크몽 가격 안내 — 디자인/문구 외주 수요는 있으나 유료 ZIP 경쟁 확인 필요.
- Canva/Miricanvas 안내문 템플릿 — 안내문 이미지 대체재가 많음.
- 소비자 상담/분쟁 콘텐츠 검색 결과 — 보상·분실 표현은 신중해야 함.

## 스타일 브리프
- 대상 독자: 동네 세탁소, 무인/셀프빨래방, 수거배달 세탁 서비스를 운영하는 소형 점주.
- 목소리: 장마철 문의가 몰리기 전에 매장 주인이 조용히 정리하는 실무 메모. 법률 자문 아님.
- 피할 것: 보상 가능/불가 단정, 소비자 권리 제한처럼 보이는 문장, 과한 매출 보장, 딱딱한 약관 말투.
- 이미지 계획:
  1. 썸네일: “장마철 세탁 문의, 접수 전에 이렇게 안내하세요”
  2. 문자 예시 카드: 이불세탁/건조 지연/이염 확인 요청
  3. 체크리스트 카드: 접수 전 확인표(소재·오염·사진·찾는 날)
- CTA: “반응이 있으면 수거배달 세탁, 운동화 세탁, 이불 보관 안내문도 따로 정리해보겠습니다.”

## 무료→유료 경로
- 무료 글: 장마철 세탁 문의 안내문과 문자 예시 8개 제공.
- 후속 ZIP: 세탁소/빨래방 접수·클레임·수거배달 안내문 60개 + 안내카드/SVG.
- 검증 기준: Tistory 검색 유입, 댓글/저장, 업종별 문구 추가 요청.
