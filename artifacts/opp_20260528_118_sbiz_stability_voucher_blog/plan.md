# 소상공인 경영안정 바우처 25만원 무료 검증 글 계획

## 결론
- 상태: scout-only `planning` GO.
- 글 각도: `25만원 받을 수 있나`보다 `신청 전 5초 판정 + 카드/사용처 실수 방지`.
- Attention Gate: 폭발 후보.

## 제목 후보
1. `소상공인 25만원 바우처, 신청 전 5초 판정부터 보세요`
2. `경영안정 바우처 25만원, 카드 선택 전에 놓치기 쉬운 것`
3. `소상공인 바우처 25만원, 예산 소진 전 확인할 4가지`

## 글 구조
1. 첫 컷: “내 사업장도 25만원 대상인가?”
2. 5초 판정표: 2025.12.31 이전 개업 / 매출 0원 초과~1억400만원 미만 / 영업 중 / 제외업종 아님 / 1인 1사업체.
3. 헷갈리는 부분: 2025년 개업자는 월평균 매출 연환산, 개업일과 발급일 혼동 금지.
4. 사용처: 전기·가스·수도, 4대 보험료, 차량 연료비, 전통시장 화재공제료. 통신비 제외.
5. 카드 선택 주의: 지급 이후 카드 변경 불가, 본인 개인카드 중심.
6. 공식 확인 순서: 전용 사이트 → 본인인증 → 사업자번호 → 카드사 선택 → 알림톡/결과 확인.
7. 마무리: 지급 보장 아님, 예산 소진/공식 공고 기준 확인.

## 이미지 계획
- 1번: 흑백 잉크 만화 썸네일. 공과금 고지서 앞 사장님 + `25만원, 내 가게도?` 말풍선.
- 2번: 4컷 판정 흐름. 매출 → 개업일 → 영업중 → 카드선택.
- 3번: 사용처/주의 행동 카드. `통신비 X`, `카드변경 X`, `12/31 잔액 회수`.

## 실행 체크리스트
- 발행 직전 공식 사이트가 신청 가능/마감인지 재확인.
- 예산 소진 여부를 단정하지 말고 공식 화면 기준으로 표현.
- GPT Image 2/코믹 스타일 이미지 3장 생성 후 로컬 QA.
- Tistory 공개 전 strict rendered QA 1회 이상, 카테고리 `돈관리·구매체크` 하위 적합 카테고리 확인.
