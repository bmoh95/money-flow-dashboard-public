# 청년미래적금 6월 갈아타기, 청년도약계좌 먼저 해지하면 안 됩니다

청년미래적금이 2026년 6월 출시 예정이라, 기존 청년도약계좌를 먼저 해지해야 하나 고민하는 분들이 있습니다.

그런데 이건 수령액 비교보다 순서가 먼저입니다.

금융위원회 4월 23일 보도자료에는 청년미래적금 출시 전 청년도약계좌를 해지하면 갈아타기 신청이 불가하다고 안내되어 있습니다.

즉, “갈아타기”를 생각 중이라면 기존 계좌부터 없애는 쪽으로 움직이면 안 됩니다.

## 먼저 볼 것만 짧게

- 도약계좌 → 미래적금 갈아타기는 2026년 6월 최초 가입 기간에 한해 허용 안내
- 공식 흐름은 신청 → 대상 통보 → 계좌개설 → 특별중도해지 → 납입 시작
- 5월 14일 기준 취급기관은 15개로 확정, 최대 금리는 기관별 7~8% 수준 안내
- 기여금은 없음, 6%, 12% 구간으로 갈림

## 갈아타기 순서는 이렇게 안내됐습니다

1. 청년미래적금 가입 신청
2. 청년미래적금 가입대상 통보 확인
3. 청년미래적금 계좌개설
4. 청년도약계좌 특별중도해지
5. 청년미래적금 납입 시작

## 5월 14일에 추가로 나온 내용

- 취급기관 총 15개 확정
- 기본금리 5% + 기관별 우대금리 2~3%p 수준, 최대 7~8% 안내
- 기관별 우대금리 세부항목은 조건별 확인 필요
- 기관별 금리 수준은 5월 말 잠정 안내 예정
