# Style brief

- 대상 독자: 노란우산공제를 “절세 된다던데 가입할까?” 수준에서 검색한 개인사업자·소상공인.
- 목소리: 세무사가 단정하는 말투가 아니라, 가입 전 메모를 같이 정리해주는 사장님 말투.
- 피할 것: 완벽 총정리, 과장된 절세액, “무조건 가입”, 유료 상품 CTA, 최근 세금 글의 `헷갈리는 것만 먼저 보기` 반복.
- 구조: 질문 3개로 판단 → 가입 전 메모 → 해지 전 멈춤 포인트 → 공식 페이지 확인 순서.
- 이미지 계획: 썸네일 1장, 가입 전 질문 카드 1장, 해지 전 체크 카드 1장. 모두 서로 다른 역할.
- CTA: 없음. 마지막에 “공식 계산/환급은 내 상황으로 다시 확인” 정도.
