# plan.md

## 제품/콘텐츠 범위
- 무료 Tistory 검증 글: 쇼핑몰·배달 소상공인 종합소득세 매출자료 수집 순서.
- 후속 가능 상품: 플랫폼별 매출/수수료/입금 대조 엑셀 템플릿 ZIP.

## 독자/구매자
- 5월 종합소득세 신고를 앞둔 스마트스토어·쿠팡·배달앱 입점 소상공인.

## 채널
- 1차: Tistory 무료 정보글.
- 2차: 검색 반응 확인 후 크몽/블로그 다운로드형 엑셀 템플릿.

## 가격/ROI 가정
- 후속 ZIP 9,900원~19,000원.
- 기준: 9,900원 x 5건 x 수수료 20% = 39,600원 순매출.
- 낙관: 19,000원 x 15건 x 수수료 20% = 228,000원 순매출.

## 차별점
- 세법 해설이 아니라 플랫폼/통장/정산 자료를 실제로 모으는 순서 중심.
- 세무 자문 리스크를 낮추고 체크리스트/파일명/자료 분류로 즉시 사용성을 확보.

## 실행 체크리스트
- 이미지 3장 생성: 썸네일, 자료 위치 카드, 누락 체크리스트.
- Tistory 렌더링에서 문단 간격·이미지 크기 확인.
- 세무 자문처럼 보이는 단정 표현 제거.
- 게시 후 유입/댓글/추가 요청 추적.
