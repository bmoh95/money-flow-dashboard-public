# 쇼핑몰·배달 소상공인 종소세 매출자료 정리 무료 검증 브리프

## 기존 작업 빠른 재점검
- MF-1 `beauty_noshow_kit`: Tistory 공개 완료 상태. 재작업/재게시 없음.
- `smartstore_review_kit`, `changelog_bounty`: 추적 상태. 새 매출/리뷰 데이터 확인 전 추가 작업 없음.
- `freelancer_simple_book_blog`, `academy_summer_class_blog`: 검증 보류. 이미지/렌더링 QA가 필요하지만 이번 실행은 신규 작업 항목 생성 우선.

## 스타일 샘플/수요 신호
- 샘플 검색: “종합소득세 매출자료 정리”, “스마트스토어 매출자료”, “배민 매출자료 종소세”, “쿠팡 매출자료 부가세/종소세”, “간편장부 매출 정리”.
- 제목 패턴: “종소세 신고 전 매출자료 확인”, “스마트스토어 매출자료 다운로드”, “배달앱 매출 누락 체크”, “간편장부 정리 순서”.
- 첫 문단: 5월 마감/자료 누락 불안에서 시작. 세무 전문용어보다 ‘어디서 뭘 내려받는지’가 핵심.
- 문단 길이: 짧은 안내 + 체크리스트/표 선호. 긴 세법 설명은 이탈 위험.
- 이미지 빈도: 홈택스/플랫폼 화면을 직접 캡처하기보다 중립 체크리스트 카드 2~3장과 흐름표가 안전.
- CTA: “자료 정리표를 따로 만들고 있다” 정도의 약한 후속 예고.

## 스타일 브리프
- 대상 독자: 5월 종합소득세 신고를 앞둔 스마트스토어/쿠팡/배민·요기요 입점 소상공인.
- 목소리: 세무사처럼 단정하지 않고, 신고 전 자료를 모으는 실무자 말투.
- 피할 것: 절세 보장, 세무 대리 행위처럼 보이는 표현, 정확한 세액 계산 약속, 플랫폼별 화면을 사칭하는 이미지.
- 이미지 계획:
  1. 썸네일: “종소세 전 매출자료, 이 순서로만 모아도 덜 헷갈립니다”
  2. 자료 위치 카드: 홈택스/스마트스토어/배달앱/PG사/통장 입금 내역 체크
  3. 누락 체크리스트: 현금영수증·카드·플랫폼 정산·수수료·환불/취소
- 소프트 CTA: “반응이 있으면 업종별 정리표도 따로 만들어보겠습니다.”

## 무료→유료 경로
- 무료 글: 신고 전 매출자료 수집 순서와 누락 체크리스트 제공.
- 후속 유료 가능성: 플랫폼별 매출/수수료/입금 대조 엑셀 템플릿 ZIP.
- 검증 기준: 5월 검색 유입, 댓글/저장/추가 요청, 후속 ‘엑셀 양식’ 검색 반응.
