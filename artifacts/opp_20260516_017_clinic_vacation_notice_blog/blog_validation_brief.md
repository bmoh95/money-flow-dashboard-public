# 동네 치과·의원 여름휴가 휴진/예약변경 안내 무료 검증 브리프

## 기존 작업 빠른 재점검
- MF-1 `beauty_noshow_kit`: Tistory 무료 공개 완료. 재작성/재게시 없음.
- 기존 launched/tracking 품목: 새 수익·리뷰·병합 신호 없음.
- 기존 redteam_hold 블로그 후보: 대부분 이미지·렌더링 QA 대기. 이번 실행은 신규 작업 항목 생성 우선.

## 이번 스카우트 후보 4개
1. GO/HOLD: 동네 치과·의원 여름휴가 휴진/예약변경 안내문 — DuckDuckGo 검색 결과에서 실제 치과 휴진 공지, 예약문자 가이드, 무료 안내문 도구/템플릿 수요 확인. 의료광고/진료 보장 표현을 피하는 무료 검증 글로 범위 제한.
2. NO-GO: GitHub/Algora 대형 인프라 바운티 재스카우트 — 공개 bounty 채널은 있으나 구현 난도·리뷰 대기·중복 PR 리스크가 크고 이번 시간대 단기 ROI가 낮음.
3. PIVOT: 병원 예약문자 유료 템플릿 직접 판매 — 구매자는 명확하나 의료기관별 개인정보/진료 안내 규정 차이가 커 유료 판매 전 무료 일반 문구 글로 검증 필요.
4. NO-GO: 여름 휴가철 병원 전화대행/ARS 세팅 서비스 — B2B pain은 있으나 외부 연락, 계정/전화 서비스 가입, 고객 응대가 필요해 사용자 부담이 높음.

## 샘플/수요 신호 14개
- DuckDuckGo `치과 여름휴가 휴진 안내 블로그`: 실제 치과 네이버 블로그/홈페이지 휴진 공지가 다수 노출.
- 검색 결과 예: “미소찬치과의원 여름휴가 휴진안내문”, “유앤이치과 2025년 여름휴가 안내”, “델타치과 8월 휴진, 여름휴가 안내”.
- 검색 결과 예: “이가영 치과 2025년 여름휴가 안내”, “연세포시즌치과 25년 여름휴가 휴진 안내 공지”, “지디스치과 24년 여름휴가 안내”.
- DuckDuckGo `동네 의원 휴진 안내문 문자 예약 변경`: 병원 예약 문자 예시, 휴진 안내문자 작성법, 병원 문자 발송 체크 콘텐츠가 노출.
- 띵톡/문자라인/모션랩스 등 병원 문자 발송 서비스 페이지가 있어 병·의원 CS 문자 시장의 상업 신호 존재.
- 예스폼/Doc4u 등 휴진 안내문·템플릿 대체재 존재.
- 연합뉴스/서울시 자료에서 휴진 정보 미고지 시 환자 헛걸음 문제가 사회적 pain으로 확인됨.
- 실제 병원 공지 글은 대부분 짧고 이미지 한 장 + 날짜/재개일/예약변경 연락처 중심.
- 병원 문자 예시 글은 “예약 확인/취소/변경/휴진 안내”처럼 상황별 문구를 모아 제공.
- 독자는 병원장보다 실무 데스크/코디네이터일 가능성이 높음.
- 의료 광고처럼 보이는 전후 효과·진료 보장 문구는 피해야 함.
- 개인정보는 예시에서 제외하고, 환자명/차트번호 같은 변수는 `{이름}` 형태로만 다뤄야 함.
- 무료 글은 휴진 공지 6문장 + 예약변경 문자 6개 + 체크리스트 정도가 적합.
- 후속 유료화는 치과/피부과/한의원/소아과별 공지·문자·ARS 멘트 ZIP으로만 검증 후 판단.

## 스타일 브리프
- 대상 독자: 여름휴가 휴진 전 공지문과 예약 변경 문자를 급히 정리해야 하는 동네 치과·의원 데스크/원장님.
- 목소리: 병원 데스크에서 실제로 예약표 보며 정리하는 사람 말투. 차분하고 과장 없음.
- 피할 것: 의료광고 문구, 진료 결과 보장, 법률 자문처럼 단정, 개인정보 예시 노출, 딱딱한 병원 행정 보고서.
- 이미지 계획:
  1. 썸네일: “여름휴가 휴진 전, 병원 안내문 체크”
  2. 문자 예시 카드: 휴진 전 예약 변경 안내 3문장
  3. 체크리스트 카드: 날짜·재개일·응급/예약변경·채널 업데이트
- CTA: “반응이 있으면 진료과별로 쓸 수 있는 문구도 따로 정리해볼게요.”

## 무료→유료 경로
- 무료 글: 휴진 공지문, 예약 변경 문자, 네이버플레이스/카카오채널/전화멘트 체크리스트 제공.
- 후속 ZIP: 치과/한의원/피부과/소아과 휴진·예약변경·검진일정 안내 문구 120개 + 카드 SVG.
- 검증 기준: Tistory 검색 유입, 병원/치과 휴진 안내문 키워드 유입, 문구 요청 댓글, 카드 이미지 클릭/저장 반응.
