# 이미지 계획

1. 썸네일/판정 카드
   - 문구: `농식품바우처 2026` / `이번 달 혜택 먼저 확인`
   - 포인트: 월 4만~18.7만원, 미사용 소멸, 사용처 확인.
   - 색감: 민트/그린, 장바구니+체크 아이콘. Pretendard Bold/SemiBold.

2. 구조 카드
   - 제목: `대상 4칸 체크`
   - 내용: 생계급여 가구 / 임산부·영유아·아동·청년 포함 / 미추진 지역 확인 / 월 금액 확인.
   - 목적: 대상 독자가 첫 화면에서 바로 걸러보기.

3. 행동 카드
   - 제목: `신청·사용 순서`
   - 내용: 공식 플랫폼 → 주소지 행정복지센터/전화/온라인 → 사용처·잔액 확인.
   - 주의: 개인정보·대행 유도 금지, 공식 링크만.
