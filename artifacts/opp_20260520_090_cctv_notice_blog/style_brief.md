# Style Brief — 매장 CCTV 안내판 3칸 체크

- 대상 독자: CCTV를 새로 달거나 기존 스티커만 붙여둔 매장·공방·학원·사무실 소상공인.
- 목소리: 겁주는 법률 글이 아니라, 문 앞 안내판을 같이 채워보는 사장님 말투.
- 피할 것: 과태료 공포 마케팅, 법률 자문처럼 단정, “완벽 준수”, 유료 템플릿/스티커 판매 CTA.
- 구조: 실수 장면 → 안내판 3칸 → 운영 중 하지 말 것 → 공식 링크 → 짧은 마무리.
- 이미지: ① 문 앞 CCTV 안내판 3칸 썸네일, ② 녹음/임의조작 금지 흐름 카드, ③ 운영관리 방침 체크카드. 서로 다른 레이아웃.
- CTA: 없음. “공식 안내와 매장 상황을 같이 보세요”로 끝낸다.
